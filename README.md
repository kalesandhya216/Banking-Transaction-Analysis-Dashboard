# Banking-Transaction-Analysis-Dashboard

This project is a Power BI dashboard that analyzes banking transaction data, customer activity, and fraud detection patterns.  
SQL was used to extract and transform data before visualization.

- Total Transactions: 3000 
- Total Customers: 500
- Total Amount: 76.27M
- Fraud Transactions: 304

- Transaction analysis by type (Credit/Debit)
- City-wise transaction analysis
- Fraud vs Non-Fraud comparison
- Monthly transaction trends
- Customer-wise analysis
